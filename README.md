# 校园碳普惠微信小程序

面向内蒙古大学校园低碳日的微信原生小程序可交互演示版。项目采用卡通绿色简笔画风格，使用内蒙古大学建筑、小山、小水和“美丽中国·我是行动者”标识。

## 运行

1. 打开微信开发者工具。
2. 导入项目目录 `D:\thxy`。
3. 编译后从首页体验；演示数据保存在微信小程序本地存储中。

## 已实现

- 首页：用户信息、空气指数、电子签到、五大专区、快捷功能入口。
- 任务：绿行专区、减塑专区、光盘专区、循环集市、低碳宿舍。
- 任务详情：自动任务、二维码任务、图片凭证和工作人员登记提示。
- 普惠商城：积分余额、兑换好礼、兑换记录和核销短码。
- 减碳排行榜：个人榜、学院榜和专区榜。
- 我的：身份资料、成就、校园集卡、活动证书和工作人员入口。
- 扩展页面：资讯中心、积分规则、校园地图、低碳课堂。

## 当前数据边界

当前版本用于 UI、流程和交互验收，签到、任务、积分、兑换、集卡和答题状态使用本地存储模拟。正式上线前需要按规格文档接入微信登录、校园身份认证、后端积分引擎、图片存储、审核核销、管理后台、数据大屏、MySQL 和 Redis。
